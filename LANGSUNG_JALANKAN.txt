JIKA ADA ERROR:

Port 5000 already in use?
→ Edit backend/.env, change PORT=5001, run RUN.bat again

MySQL not found?
→ Open MySQL Workbench, copy setup-database.sql content, execute

npm install hangs?
→ Press Ctrl+C, run: npm cache clean --force, run RUN.bat again

Node.js not found?
→ Download & install from https://nodejs.org/
