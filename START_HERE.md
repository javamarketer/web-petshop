## 🚀 Development Quickstart (3 Steps)

### Step 1: Setup (10 minutes)
```bash
# Backend
cd backend
npm install
cp .env.example .env
npm run migrate
npm run dev

# Frontend (new terminal)
cd frontend
npm install
npm run dev
```
